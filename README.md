## Installation

- `git clone https://github.com/roger3g/facebook-clone.git`;
- `cd facebook-clone`;
- `npm install`;
- `npm run dev`.

Requirements to run **locally**:

- [ ] [Node.js](https://nodejs.org/en/) ( Required )
- [ ] [GIT](https://git-scm.com/) ( Optional )
